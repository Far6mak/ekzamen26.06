# Итог по заданию FizzBuzz (13 challenges)

## Описание программы
Программа реализует задачу FizzBuzz: для чисел в диапазоне (обычно 1–99) выводится:
- `fizz` — если число делится на 3
- `buzz` — если делится на 5
- `fizzbuzz` — если делится и на 3, и на 5

Реализация встречалась в разных вариантах: функции, классы, работа с файлами и модулями.

---

## Что было исправлено
В ходе выполнения заданий были найдены и исправлены ошибки:
- синтаксические ошибки (кавычки, двоеточия, отступы)
- ошибки типов данных (str вместо int)
- логические ошибки условий и порядка `if/elif`
- ошибки диапазона `range()`
- ошибки вызова функций
- FileNotFoundError при работе с файлами
- ошибки импорта модулей
- ошибки в ООП (self, методы класса)
- опечатки и неверные имена функций/переменных

---
